Select an X3D file to load (.PNG textures will be automatically searched in its folder):



  • A = Full model view
  • W = Walk mode (with mouse)
  • E = Examine mode (rotate around origin)
  • I = "Look at" mode
  • R = Reset view
  • O = Strange view mode
  • D = Debug mode
  • G = Immediate pan mode
  • F = flight mode
  • U = Reset flight mode
